# 🚀 Welcome to Truck DMS!

We're excited to announce the launch of **Truck DMS** - a comprehensive document management system designed specifically for trucking companies.

## ✨ What's New

### Complete Document Management
- **7 Document Types**: RC, BOL, POD, Dispatcher, Fuel, Invoice, RLP
- **Smart Filtering**: Filter by type, file format, and search terms
- **Export Capabilities**: PDF and Excel export with custom formatting
- **Modern UI**: Clean, responsive interface with real-time updates

### Easy Setup
- **One-Click Installation**: Windows batch file handles everything
- **Automatic Dependencies**: Installs Python, packages, and creates environment
- **Desktop Integration**: Creates shortcuts for easy access
- **Cross-Platform**: Works on any system with Python 3.8+

## 🎯 Key Features

### Document Types Supported
| Type | Use Case | Key Features |
|------|-----------|--------------|
| **Rate Confirmation** | Load planning | Load numbers, dispatcher info, mileage tracking |
| **Bill of Lading** | Shipping docs | BOL numbers, load details, route info |
| **Proof of Delivery** | Delivery confirmation | Sign-off, delivery details, timestamps |
| **Dispatcher Management** | Commission tracking | Company info, contact details, payment tracking |
| **Fuel Receipts** | Expense tracking | Receipt numbers, amounts, tax reporting |
| **Invoicing** | Billing management | Invoice numbers, payment tracking, quick pay |
| **Rate Lock Protection** | Financial protection | RLP numbers, amount received, date tracking |

### Export Options
- **PDF Reports**: Professional formatted documents with company branding
- **Excel Workbooks**: Multi-sheet exports with summary data
- **Filtered Export**: Export only selected documents
- **Custom Fields**: Include all document-specific data

## 🛠️ Technical Highlights

### Frontend

### Deployment

- **Documentation**: Comprehensive setup and usage guides
- **Cross-Platform**: Windows batch files for easy setup

## 🚀 Getting Started

### Quick Start (Windows)
1. **Download** the project files
2. **Run** `start_dms_simple.bat`
3. **Wait** for automatic setup
4. **Access** at `http://localhost:5000`

### Manual Setup
1. **Install Python 3.8+** from [python.org](https://www.python.org/)
2. **Clone** this repository
3. **Install dependencies**: `pip install flask pandas openpyxl xlrd xlsxwriter`
4. **Run**: `python app.py`
